## Додатковий етап: виправлення сортування і page navigation

Після виявлення помилок у роботі виконано окремий етап виправлень.

### Виправлені проблеми

- некоректне сортування каталогів і сторінок з числовою нумерацією та початковими пробілами в іменах
- нестабільна робота кнопок `Попередня/Наступна`

### Що змінено

- у `services/fswalker.py` додано нормалізацію sort key через обрізання крайніх пробілів
- сортування сторінок переведено на `path_sort_key()` по сегментах шляху
- додано `get_page_navigation()` для отримання `prev/next`
- у `routes/main.py` додано `/api/navigation`
- у `static/js/tree.js` кнопки навігації переведено на server-side navigation context

### Що додано в тести

- перевірку `prev/next` через `/api/navigation`
- окремі тести на природне сортування для кейсу з російськими каталогами і початковими пробілами

### Результат перевірки

- усі тести проходять
- сортування `0.0.1.1 ... 0.0.1.11` перевіряється автоматично
- navigation route повертає коректні `prev/next`
