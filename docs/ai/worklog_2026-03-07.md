## Перевірка працездатності

### Автоматизовані тести

Запущено:

```bash
./.venv/bin/python -m unittest discover -s tests -v
```

Результат:

- `5` тестів виконано
- усі тести успішні

### Функціональна перевірка

Додатково перевірено:

- `/api/folder?path=..` повертає `403`
- `/book/../app.py` повертає `403`
- `/` повертає `200`

---

## Локальний доступ для перевірки

### Запуск

```bash
source .venv/bin/activate
python app.py
```

### Адреса

```text
http://127.0.0.1:5000
```

або

```text
http://localhost:5000
```

### Поточний статус

Сервер запущено для ручної перевірки.

Додатково підтверджено HTTP-відповідь:

- `GET /` -> `200 OK`

Для підняття локального сокета в цьому середовищі запуск було виконано поза sandbox, бо sandbox блокує bind на порт.
